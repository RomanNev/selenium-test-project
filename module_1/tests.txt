Тестовый сценарий 1.1.1
Добавление товара с главной страницы каталога
Предусловия:
1. Открыть сайт http://selenium1py.pythonanywhere.com/ru/catalogue/
2. Корзина товаров пуста 
Шаги:
1. Положить любой товар в корзину.
ОР: 
- Появляется уведомление об успешном добавлении товара в корзину.
- Появляется уведомление с общей стоймостью корзины, она совпадает со стоймостью товара, в нем есть кнопки "просмотреть корзину" и "оформить".
- В шапке сайта, рядом с кнопкой "просмотреть корзину", отобразилась цена добавленного товара. 

2. Перейти в корзину через появивщееся уведомление, кликнув по кнопке "просмотреть корзину".
ОР: 
- Открывается корзина с выбранным товаром. 
- Цена в корзине совпадает со стоймостью товара в каталоге.
- есть сообщение о наличие товаров в корзине



Тестовый сценарий 1.2.1
Поиск товара по каталогу
Предусловия:
1. Открыть сайт http://selenium1py.pythonanywhere.com/ru/catalogue/
Шаги:
1. Ввести в поиск "Learning Python"
ОР: 
- текст вводится в поисковой инпут.
2. Кликнуть кнопку "найти".
ОР: 
- в выдаче присутувует товар с названием "Learning Python".
3. Стереть введенный текст из первого шага 
ОР: 
- текст корректно удаляется из инпута
4. Ввести в поиск "Python" и кликнуть кнопку "найти".
ОР:  
- Текст вводится в поисковой инпут.
- В выдаче присутувует товар с названием "Learning Python".


Тестовый сценарий 2.1.1
Регистрация нового пользователя с главной страницы
Предусловия:
1. Открыть страницу http://selenium1py.pythonanywhere.com/ru/accounts/login/
2. Пользователь не авторизован 
Шаги:
1. В блоке "Зарегистироваться" ввести корректный email в поле "Адрес электронной почты".
ОР: 
- текст корректно вводится в инпут "Адрес электронной почты".
2.В блоке "Зарегистироваться" ввести корректный пароль в поле "Пароль" и "Повторите пароль".
ОР: 
- Тектс корректно вводится в инпуты.
3. Кликнуть на кнопку "Зарегистироваться".
ОР: 
- Появляется страница с сообщением "спасибо за регистацию".
- Пользователь зарегистирован.

-- Тестовый сценарий 3.1.1
Удаление продукта из корзины
Предусловия:
Открыта старница корзины, в корзине есть добаввленный продукт
Шаги добавление товара
1. Открыть сайт  http://selenium1py.pythonanywhere.com/ru/
2. Перейти в раздел "все товары"
3. Добавить любой продукт в корзину
4. Перейти в корзину
Шаги:
1. Тапнуть на кнопку "удалить" в блоке существующего товара
ОР:
- товавр удален из корзины
